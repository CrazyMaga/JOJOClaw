# 模拟交易技能

本技能用于A股模拟交易，支持买入、卖出、查询资产、查看交易记录功能。

## 交易流程

执行交易时自动完成以下8个步骤：

### 步骤1: 交易时间检查
- 集合竞价：9:15-9:25
- 连续竞价：9:30-11:30、13:00-14:57
- 非交易时间无法交易（测试模式可跳过）

### 步骤2: 获取实时股价
- 自动从腾讯财经获取实时行情
- 获取当前价和昨收价

### 步骤3: 涨跌幅限制检查
- 主板：±10%
- 创业板/科创板：±20%
- ST股票：±5%
- 超出涨跌停价无法交易

### 步骤4: 交易单位检查
- 主板/创业板：买入100股整数倍
- 科创板：200股起，可按1股递增
- 卖出可为零股

### 步骤5: 资金/持仓/T+1检查
- 买入：检查可用资金是否充足
- 卖出：检查持仓数量是否足够 + T+1制度检查（当日买入需下一交易日才能卖出）

### 步骤6: 执行交易
- 买入/卖出操作

### 步骤7: 费用计算
- 佣金：万分之2
- 印花税：卖出时千分之1
- 过户费：沪市万分之0.1，深市含在佣金中

### 步骤8: 更新资产和交易记录
- 更新总资产、可用资金
- 记录交易明细

## 数据文件

交易数据存储在 `data/trading.json`，包含：
- `total_assets`: 总资产（元）
- `available_cash`: 可用资金（元）
- `positions`: 持仓记录
- `transactions`: 交易记录

## 操作命令

```bash
# 买入股票（自动获取实时价格）
python scripts/trading.py buy <股票代码> <数量>

# 买入指定价格
python scripts/trading.py buy <股票代码> <数量> <价格>

# 卖出股票（自动获取实时价格）
python scripts/trading.py sell <股票代码> <数量>

# 卖出指定价格
python scripts/trading.py sell <股票代码> <数量> <价格>

# 查询实时股价
python scripts/trading.py price <股票代码>

# 查询资产状况
python scripts/trading.py status

# 查看交易记录
python scripts/trading.py history
```
